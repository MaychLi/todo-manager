## ✨ Key Features

* **Full CRUD Operations:** Seamlessly create, read, update, and delete tasks.
* **Smart Filtering:** Instantly switch between task views (All, Active, Completed).
* **Responsive Design:** A mobile-first approach ensuring a flawless experience across all devices.
* **Advanced Input Handling:** Custom multiline input support that automatically adapts its layout without breaking the UI.
* **Test Coverage:** Core business logic and key UI components are covered by unit tests to ensure reliability.

## 🛠 Tech Stack
*  **Architecture:** Feature-Sliced Design
* **Frontend:** React 18, TypeScript
* **UI Library:** Material-UI (MUI) 
* **State Management:** Redux Toolkit
* **Testing:** Jest, React Testing Library
* **Styling:** MUI prop, CSS Modules

## 🧠 Architecture & Highlights
> In this project, I focused on a component-driven approach and the Separation of Concerns principle. State management is delegated to Redux, keeping the React components declarative, predictable, and easy to test. 
> Special attention was given to heavily customizing Material-UI components to achieve a unique, non-generic dark mode aesthetic while maintaining strict accessibility standards.
